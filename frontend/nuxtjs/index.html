# 使用模板快速开始

# 根据你的需求进行选择
yarn create nuxt-app nuxt-test

# 使用想法

NuxtJS 是一个重约定的框架。

# components

可以有目录结构：

components
├── Component-1.vue
└── Menu
    └── Component-2.vue

引入时：

<MenuComponent-2 />

# layouts

可以有目录结构：

layouts
└── 10
    └── 1
        └── default.vue

引入时：

export default {
	layout: '10/1/default'
}

# pages 动态路由

使用下划线为动态路由：

pages
├── games
│   └── _module
│       └── _id.vue

相当于 /games/:module/:id。
